# Student Marks Management System

## Overview

The Student Marks Management System is a Python application designed to manage student academic records and attendance efficiently. The system stores data from different assessments, tracks attendance, calculates performance statistics, and generates final results for students.

This project was developed to strengthen our Python programming skills and gain practical experience in data management and software development.

## Features

* Manage student grades from multiple assessments
* Record and update quiz marks
* Record and update lab marks
* Track lecture attendance
* Track lab attendance
* Calculate student performance statistics
* Generate final results
* Identify highest, lowest, and average scores
* Modify attendance and grades when necessary

## Project Files

| File                 | Description                              |
| -------------------- | ---------------------------------------- |
| `project.py`         | Main program containing the system logic |
| `Marks`              | Stores student marks                     |
| `Grades`             | Stores calculated grades                 |
| `Quizzes`            | Stores quiz scores                       |
| `Lab_Marks`          | Stores laboratory assessment marks       |
| `Lecture_Attendance` | Stores lecture attendance records        |
| `Lab_Attendance`     | Stores laboratory attendance records     |
| `Finalresult`        | Stores final student results             |

## Requirements

* Python 3.x

## Running the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/student-marks-management-system.git
```

2. Navigate to the project folder:

```bash
cd student-marks-management-system
```

3. Run the program:

```bash
python project.py
```
